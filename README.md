**项目说明**

基于 [Think PHP6.0](https://www.thinkphp.cn/)，[Layui](https://layuion.com/)，[Pear Admin](http://www.pearadmin.com/) 实现的留言板系统

**运行环境**
PHP8.0 + MySQL5.7

**安装教程**

1. 导入根目录的 data.sql 文件
2. 打开 config/database.php
3. 设置数据库名，数据库用户名，数据库密码
